# SALARY--SAVVY
Developed a flask-based web application that analyses salary and expenses to provide data driven budgeting and savings recommendations.

Salary Savvy is a Data Science-based web application that helps users manage their salary, track expenses, and improve savings using smart insights and visualizations.

---

## Features

- Salary input system  
- Expense tracking by category  
- Data visualization using charts  
- Smart financial suggestions  
- Multilingual support (English, Hindi, Telugu)  
- Sparky Robo Assistant for guidance  

---

## Project Objective

The main aim of this project is to apply data science concepts in real-life financial planning.  
It helps users understand their spending patterns and make better budgeting decisions.

---

## Tech Stack

- Python (Flask)  
- HTML, CSS, JavaScript  
- Chart.js  

---

## How It Works

1. User enters salary details  
2. Adds expenses in different categories  
3. System analyzes the data  
4. Visual charts are generated  
5. Smart suggestions are provided to improve savings
